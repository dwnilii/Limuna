[🇺🇸 English](README.md)

# معرفی

لیمونا یک پنل مدیریت سرور لینوکس است که به صورت کامل در **User Space** اجرا می‌شود و از طریق **SSH** به سرورهای مقصد متصل می‌شود.

بر خلاف بسیاری از پنل‌های مشابه، برای مدیریت سرور **نیازی به نصب هیچ Agent یا Daemon روی سرور مقصد وجود ندارد.**

لیمونا با نگه داشتن اتصال SSH و کش کردن درخواست‌های فقط خواندنی، مصرف منابع سیستم و تعداد پردازش‌های SSH را به حداقل می‌رساند.

# نصب

پروژه را داخل مسیر `/opt` کلون کنید.

```bash
cd /opt

git clone https://github.com/dwnilii/Limuna.git limuna

cd limuna

npm install

npm run build

npm install -g pm2

pm2 start ecosystem.config.cjs
```
